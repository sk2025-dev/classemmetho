```jsx
<img src={getPhotoUrl(user.photo_path, user.nom, user.prenom)} />
```

2️⃣ **Comment uploader?**
```jsx
<PhotoUploadInput onPhotoSelected={(url) => setPhoto(url)} />
```

3️⃣ **Où copier les fichiers?**
